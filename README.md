# JavaScript Operators Demo

This project demonstrates basic **JavaScript concepts** including:
- Variables (`const`, `let`)
- Data types
- Arithmetic operators
- Increment operators
- Bitwise operators

---

project-folder

---

##  Features

### 1. Variables
- `const` → cannot be reassigned  
- `let` → can be reassigned  

### 2. Data Types
- Use `typeof` to check variable type

### 3. Arithmetic Operators
- Addition `+`
- Subtraction `-`
- Multiplication `*`
- Division `/`
- Modulus `%`
- Power `**`

### 4. Increment Operators
- `a++` → post-increment  
- `++a` → pre-increment  

### 5. Bitwise Operators
- AND `&`
- OR `|`
- NOT `~`
- XOR `^`
- Left Shift `<<`
- Right Shift `>>`
